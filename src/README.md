# Switch
## Slots
| Name | Type | Default | Description |
| :-----------: | :-----------: | :-----------: | :-----------: |
| value | Boolean | false | 当前开关是否选中 |
| showText | Boolean | true | 是否显示当前开关文字 |
## Events
| Name | Description | Return |
| :-----------: | :-----------: | :-----------: |
| bind:change | 开关变化时触发， 返回当前值 | { value }
